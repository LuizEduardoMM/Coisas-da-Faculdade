Trabalho para Estrutura de Dados, uma lista encadeada de strings puxadas de um arquivo. 
No código os args[] na main são para poder ser rodado no cmd usando quaisquer arquivos, 
sendo eles na ordem entrada e saída. Poderia ter feito duplamente encadeada para ser mais 
fácil encontrar o anterior, mas quis esse desafio a mais. No caso de uma request ser inválida 
(formatação tem de ser padrão, com ADD para adicionar, SHOW para listar e REMOVE para remover), 
eles darão o aviso no arquivo saída, o mesmo para se forem vaíldas e ambas formatadas na saída 
para se manter esse padrão.
